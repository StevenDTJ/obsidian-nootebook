# 一、常见的气体
### 1. 气体
氮气（78%）、氧气（21%）、稀有气体（0.94%）、二氧化碳（0.03%）、其他气体及杂质（0.03%）
### 2. 天然气和沼气
主要成份就是甲烷。甲烷无毒，但有窒息作用。当其甲烷在空气中浓度达到 10%时，可使人窒息死亡。
### 3. 一氧化碳
人工煤气的主要成分。不溶或仅微溶于水，所以在**煤灶上放水不能防止煤气中毒**
### 4. **二氧化碳**
不支持燃烧（用途：灭火），不能供给呼吸（为呼出气体的主要成分）；
干冰（二氧化碳的固体形态），用于制冷和人工降雨
### 5. 氧气
供给呼吸（如供潜水、医疗急救）；支持燃烧（如燃料燃烧、炼钢）
燃烧条件：达到着火点；有助燃剂。
灭火时**热水比冷水效果好**（热水形成蒸汽，隔绝氧气）
### 6. 氮气
惰性保护气（化学性质不活泼）、重要原料（硝酸、化肥）、液氮冷冻
### 7. 稀有气体
指氦、氖、氩、氪、氙、氡等，反应性很低。可作为保护气、电光源等。

# 三、酸、碱、盐
1. 浓盐酸：有刺激性气味，无色液体，一般用于金属除锈，制造氯化物等。
2. NaOH 氢氧化钠：又叫烧碱、火碱、苛性钠。在空气中容易潮解，强腐蚀性。固体干燥剂。
3. Ca(OH)2 氢氧化钙:又叫熟石灰、消石灰，白色粉末，微溶于水，弱腐蚀性。
4. NaCL：俗称食盐，可用于食用，消除积雪等。
5. Na2CO3碳酸钠：纯碱，用于玻璃、造纸、纺织、洗涤、食品工业等。
6. 化学肥料（氮肥、钾肥、磷肥）
	1. 氮肥：**缺氮导致叶黄**。生物固氮：豆科植物的根瘤菌将氮气转化为含氮的化合物而吸收。
	2. 钾肥：**缺钾会导致叶尖发黄**。草木灰：农村最常用钾肥（主要成分为K2CO3），呈碱性。
	3. 磷肥：**缺磷**会导致**生长迟缓，产量降低，根系不发达**。

# 四、化学生活小常识
1. 人们经常用铁做炒锅，而不是铜或铝。
2. 石灰岩简称灰岩，是以**方解石**为主要成分的**碳酸盐岩**。石灰岩是生产水泥的主要材料。
3. 半导体指常温下导电性能介于导体与绝缘体之间的材料，而石英正是具有这样的特性，因此石英可以用于制作半导体。
4. 稀土(rare earth)有“工业维生素”的美称。
5. 隐形飞机机身材料和涂层对雷达发射的电磁波具有良好的吸收作用，并且机身特殊的形状也减弱电磁波的反射，防止被雷达发现。
6. 鳄鱼流泪的原因是**鳄鱼的肾脏发育不完全**，需要靠眼睛附近的腺体排除盐分。
7. 碘盐的储存和食用：盛放碘盐的器皿的应为**棕色遮光**的瓶罐；应放在阴凉，干燥处避免受日光直射和吸潮；存放时间不宜过长。
