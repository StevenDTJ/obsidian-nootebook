
## 4.1 
**一个符号两种条件**
1. 充分条件（箭头前面）
2. 必要条件（箭头后面）
- 必要条件关键词：基础、关键、前提、需要、必须、离不开、不可或缺

**四种相对关系**
1. 和且关系：“A 且 B ”的否定是“非 A 或非 B”，“A 或 B ”的否定是“非 A 且非 B”，
2. 如果那么/只要就：“如果 A 那么 B”等于“A → B”
3. 只有才：“只有 A 才 B”等于“B → A”，等于“不 A 就不 B”
4. 除非否则：“必须 A 否则 B”，等于“除非 A 否则 B”，等于“非 A → B ”

**常见提示词**

|        常见提示词        |   翻译为    |
| :-----------------: | :------: |
| A 并且 B、A 和 B、A 同时 B |  A 且 B   |
|    A 或 B、AB 至少一个    |  A 或 B   |
|     AB 之多一个，翻译为     | 非 A 或非 B |
|    非（A 且 B），翻译为     | 非 A 或非 B |
|    非（A 或 B），翻译为     | 非 A 且非 B |
