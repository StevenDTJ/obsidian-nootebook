# 二、夏商周时期


## （一）夏朝

1. 禹传子启，家天下，**世袭制代替禅让制**，定都阳城，最早的奴隶制国家
2. 二里头文化（最早的宫殿遗址）
3. **夏历：《夏小正》**
4. 商汤灭夏

夏朝的最后一个国君夏桀无道，失去民心，商部落的首领商汤乘机举兵伐夏，推翻了夏朝的统治。


> [!important] 鸣条之战：商汤灭夏桀


## （二）商朝

1. 青铜器：后母戊鼎，我国迄今考古发现的最大青铜器
2. 甲骨文：刻写在龟甲和兽骨上，是汉字的源头，甲骨文之父——王懿荣
3. 盘庚迁殷：曾经五次迁都，直到盘根迁都于殷（今河南安阳）
4. 商的灭亡：牧野之战

    宗法制：嫡长子继承制
2. 国人暴动
3. 灭亡：周幽王烽火戏诸侯

# 三、秦汉时期


## （一）秦朝（第一个统一的中央集权封建国家）


### 1. 统一全国

- 政治上：三公九卿制（丞相、太尉、御史大夫），地方实行郡县制
- 经济上：统一货币度量衡
- 思想上：焚书坑儒，统一文字（小篆）
- 军事上：北击匈奴，修建长城，开凿灵渠等

> [!important] 沙丘之变：秦始皇去世后，李斯和赵高发动政变


> [!important] 我国古代四大民间传说：孟姜女、牛郎织女、梁祝、白蛇传


### 2. 大泽乡起义


公元前 209 年，陈胜、吴广在大泽乡举起了中国历史上**第一次大规模农民起义**的旗帜，喊出了“王侯将相宁有种乎”的口号

> [!important] 垓下之战：楚汉决战

东汉时期，班超帮助西域诸国大败匈奴，疏通了西域南道。公元 91 年，班超被任命为西域都护，驻守龟兹，打通西域北道，班超被封为定远侯。

# 五、隋唐时期


## （一）隋朝


### 1. 开皇之治


隋文帝杨坚在位时期社会民生富庶、人民安居乐业、政治安定，史称“**开皇之治**”。


> [!important] 典故：一衣带水


### 2. 开通大运河


隋炀帝为了便利漕运和军事运输，利用天然河流与旧有渠道，开凿了一条以洛阳为中心，沟通南北的大运河。大运河共四段:永济渠、通济渠、邗沟、江南河。
大运河：隋唐大运河、京杭大运河、浙东大运河

1. 春秋：夫差开凿邗沟
2. 隋朝，以洛阳为中心，永济渠、通济渠、邗沟、江南河，沟通渭河、黄河、淮河、长江、钱塘江五大水系
3. 元朝：截弯取直，京杭大运河（世界上最长的人工运河）

## （二）唐朝


### 1. 玄武门之变


626 年，秦王李世民在长安宫城的玄武门发动政变，杀掉太子李建成和齐王李元吉。


### 2. 贞观之治


由于唐太宗知人善用、虚心纳谏，采取了轻徭薄赋、厉行节约等政策，并大力平定外患。稳定边疆。使社会出现了安定、清明的政治局面


### 3. 开元盛世


唐玄宗在位前期（开元年间），政治比较清明，经济迅速发展，提倡文教，使得天下大治，唐朝进入全盛时期，成为当时世界上最强盛的国家，史称“开元盛世”。


### 4. 安史之乱


安史之乱是指安禄山、史思明发动的叛乱，是唐朝由盛而衰的转折点。


### 5. 黄巢起义


黄巢起义直接打击了唐王朝的统治，加速了唐朝的灭亡。


> [!important] 冲天香阵透长安，满城尽带黄金甲


### 6. 相关制度

1. 科举制度
    隋文帝——分科考试
    隋炀帝——进士科，考察对时事的看法
    武则天——殿试和武举
2. 三省六部制
    **三省**：
    - 尚书省：最高行政机构，负责执行国家的重要政令
    - 门下省：审议机构，负责审议政令
    - 中书省：决策机构，负责草拟和颁发皇帝的政令
        尚书省下设**六部**，分别为为
    - 吏部：负责考核、任免四品以下官员
    - 户部：负责财政、国库
    - 礼部：负责贡举、祭祀、典礼
    - 兵部：负责军师
    - 刑部：负责司法、审计
    - 工部：负责工程建设
        **五官**：
    - 司马：军事
    - 司空：水利、工程
    - 司徒：民政、户籍
    - 司士：纠察百官
    - 司寇：刑狱

# 六、宋元时期


## （一）宋朝


### 1. 陈桥驿兵变


建立北宋，定都汴京（今开封），史称宋太祖


### 2. 杯酒释兵权


北宋初年，宋太祖为了巩固统治，设酒宴，解除了石守信等人的兵权


### 3. 檀渊之盟


澶渊之盟签订于宋真宗时期，是北宋与辽经过多次战争后所缔结的一次盟约。盟约缔结后。宋、辽之间百余年间不再有大规模的战事。


### 4. 靖康之变


1127 年，金国掳走徽、钦二帝，北宋灭亡


### 5. 南宋建立


1127 年，赵构在**应天府南京**（**今河南商丘**）称帝，后来定都临安（今杭州），史称南宋


### 6. 岳飞抗金


南宋初年，金军大举进攻南宋，岳飞组织岳家军，在**郾城**大败金军，权臣秦桧以“莫须有”的谋反罪名杀害岳飞


### 7. 绍兴和议


1141 年，宋金形成南北对峙


### 8. 文天祥抗元

# 七、明清时期


## （一）明朝


### 1. 建立


1368 年，朱元璋率领农民起义，推翻了元朝政权，建立了明朝，即明太祖。明初定都南京，后来，明成祖朱棣迁都北京


### 2. 政治制度

1. 在中央：废丞相，权分六部
2. 在地方：三司分权，都指挥司、承宣布政使司、提刑按察使司，分主军事、民政、司法。改大都督府为立军都督府，统兵权和调兵权分离。
3. 厂卫特务机构（东厂、西厂、锦衣卫）
4. 实行八股取士，加强思想控制

### 3. 经济与对外交流

1. 张居正“一条鞭法”改革：把田赋、徭役和杂税合并起来，**折成银两**，分摊到田亩上，按田亩多少收税
2. 郑和下西洋：明成祖时期，郑和于 **1405-1433** 年七下西洋，从刘家港出发，最远到达非洲东海岸和红海沿岸
3. 农作物：原产美洲的甘薯、玉米、马铃薯、烟草传入中国
4. 科技：利玛窦带来西方的科技著作，徐光启《泰西水法》引入欧洲水利技术
5. 海防：明朝的戚继光抗倭，郑成功从荷兰殖民者手中收复台湾。

## （二）清朝


### 1. 建立

- 1616 年，女真首领努尔哈赤建立“后金”
- 1636 年，**皇太极**在沈阳称帝，改女真族为满族，改国号为“清”
- 1644 年，李自成领导的农民军攻破北京城，结束了明朝的统治。驻守山海关的明朝将领吴三桂引清军入关，击败农民军，后来清军逐步统一全国，定都北京。

### 2. 巩固发展

1. 康熙：平定**三藩之乱**（平西王吴三桂、平南王尚可喜、靖南王耿精忠）；灭台湾郑氏政权，设立台湾府；两次击退盘踞雅克萨的俄军，签订《尼布楚条约》
2. 雍正：实行密折制度，设立军机处（**标志君主专制达到顶峰**）；雍正全面实施“改土归流”制度，革除土司制度；经济上实行摊丁入亩（**废除人头税，按田地面积征收**）
3. 乾隆：平定天山南路的大小和卓叛乱，改西域为新疆；1762 年，清朝设伊犁将军，统管新疆地区；加强对西藏地区管理，实施金瓶掣签制度。

### 3. 重农抑商和海禁政策

1. 明清时，积极推行“重农抑商”政策，严格限制商人活动
2. 强行“海禁”政策
