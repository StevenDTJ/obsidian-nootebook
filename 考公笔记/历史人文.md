# 二、夏商周时期
## （一）夏朝
1. 禹传子启，家天下，**世袭制代替禅让制**，定都阳城，最早的奴隶制国家
2. 二里头文化（最早的宫殿遗址）
3. **夏历：《夏小正》**
4. 商汤灭夏
夏朝的最后一个国君夏桀无道，失去民心，商部落的首领商汤乘机举兵伐夏，推翻了夏朝的统治。
> [!important] 鸣条之战：商汤灭夏桀
## （二）商朝
1. 青铜器：后母戊鼎，我国迄今考古发现的最大青铜器
2. 甲骨文：刻写在龟甲和兽骨上，是汉字的源头，甲骨文之父——王懿荣
3. 盘庚迁殷：曾经五次迁都，直到盘根迁都于殷（今河南安阳）
4. 商的灭亡：牧野之战
> [!important] 隶书是古代汉字的分水岭

## （三）西周
1. 西周实行分封制、宗法制和井田制
	分封制：天子——诸侯——卿大夫——士大夫
	宗法制：嫡长子继承制
2. 国人暴动
3. 灭亡：周幽王烽火戏诸侯
## （四）东周
### 1. 春秋
典故：
1. 齐桓公是春秋第一个霸主
	典故：管鲍之交、长勺之战、尊王攘夷、老马识途
2. 晋文公：退避三舍（退避三舍）、秦晋之好、寒食节的由来
3. 楚庄王：一鸣惊人、问鼎中原、饮马黄河
4. 勾践：卧薪尝胆、鸟尽弓藏、兔死狗烹
### 2. 战国
- 战国七雄：齐楚燕韩赵魏秦
- **主要战争**
	- 桂陵之战——齐魏——围魏救赵
	- 马陵之战——齐魏——减灶诱敌
	- 长平之战——秦赵——纸上谈兵
- 主要变法
	- 秦孝公任用商鞅实行变法。相关典故：**徙木立信、作法自毙**等。
		- 经济上——废除井田制，实行土地私有；实行重农抑商政策；统一度量衡
		- 政治上——编制户籍，连坐；推行**县制**
		- 军事上——奖励军功
	- **李悝变法**（魏国）：《法经》（第一部比较系统的地主阶级法典）；**平籴法**（政府平抑粮价物价）
- 战国四君子
	- 信陵君：窃符救赵
	- 孟尝君：鸡鸣狗盗
	- 平原君：利令智昏、毛遂自荐、脱颖而出、一言九鼎
	- 春申君：移花接木
> [!important] 记忆方法：信服——窃符；尝——鸡狗；春——花木
- 其他
	- 水利：秦国时期修筑都江堰：李冰父子
	- 科学成就
		- 战国时期发明“司南”，世界最早的指南工具
		- 战国时期甘德、石申二人测定的恒星记录《甘石星经》，世界最早
		- 《黄帝内经》，记载了我国最早的人体解剖知识，我国现存最早的医书
> [!important] 中医四大经典：《黄帝内经》、《难经》（扁鹊）、《伤寒杂病论》（张仲景）、《神农本草经》

> [!important] 中医别称：岐黄、青囊、杏林、悬壶 
# 三、秦汉时期
## （一）秦朝
### 1. 统一全国
- 政治上：三公九卿制，地方实行郡县制
- 经济上：统一货币度量衡
- 思想上：焚书坑儒，统一文字（小篆）
- 军事上：北击匈奴，修建长城，开凿灵渠等
