一、关键信息
 
| 关键信息      | 概念                           | 常见特征词                 |
| --------- | ---------------------------- | --------------------- |
| **1. 主体** | 某种行为或事件的发起者                  | ……是（动词）               |
| **2. 客体** | 与主体相对，某种行为或事件的承受者、被指向者       | 对于……、对……、（动词）……       |
| **3. 方式** | 主体为达到目的所采用的方法和措施             | 以……方式、……手段、通过……       |
| **4. 结果** | 行为或者事件的最终状态                  | 导致……、造成……、最终……        |
| **5. 原因** | 主体发出动作的理由，原因一定发生在结果之前        | 由于……、由……导致、……引起       |
| **6. 条件** | 一般指达到结果的必要条件，比如特定的时间、地点或其他限制 | 以……为基础、在……前提下、在……条件下  |
| **7. 目的** | 主体预先设想的行为目标和结果               | 以……为目的、为了……、确保……、出于…… |
| **8. 本质** | 事物本身所固有的根本的属性                | ……一种……                |

二、选项比较
1.要选择：相对比较更好的，更贴合题干信息的。
![[Pasted image 20260307151244.png]]
【例题】B 比 C 好，因为 C 不是“非爱性 的”

2.当难以比较的时候，可以看哪个选项出现了题干中的原词或近义词来判断
![[Pasted image 20260307151427.png]]【例题】C 对，因为包含题干原词“灵感”

3.排除同构选项：可以排除比较相似的选项
![[Pasted image 20260307152123.png]]
