# 图形推理 - 样式类

## 一、样式类概述

样式类规律是图形推理的核心考点之一，主要考查图形的叠加、变换和运算能力。

## 二、题型判定

### 2.1 识别特征

**出题特点：** 元素组成**相似**
- 并非完全不同
- 也非完全相同

**判定口诀：**（参见 [[图推题目类型判定技巧]]）
> **元素组成相似，看样式变化**

### 3.2 加法

#### 3.2.1 元素相加

| 类型 | 说明 |
|:-----|:-----|
| 全部相加 | 所有元素叠加 |
| 部分相加 | 部分元素叠加 |
| 覆盖 | 元素覆盖叠加 |

#### 3.2.2 颜色加法（黑白运算）⭐高频考点

**图形特点：**
- 有黑有白
- 黑块数量不同
- 黑白数量差值不大（小于3）

**做题方法：**
> 相同位置处，前两图颜色相加等于第三图颜色

**颜色运算规则：**
- 白 + 黑 = 黑
- 黑 + 白 = 黑
- 白 + 白 = 白
- 黑 + 黑 = 白

（或根据具体题目规则而定）

![[image/Pasted image 20260303172625.png]]

> **上图说明：** 白+黑、黑+白的结果不同

### 3.3 求同

#### 3.3.1 去异求同

**定义：** 前两个图形叠加在一起，**取重叠的部分**

#### 3.3.2 整体求同

**定义：** 一行/列图形中，**所有图形都具有的共同元素**

**例图：**
- 第一行都有 **圆**
- 第二行都有 **正方形**

### 3.4 求异

#### 3.4.1 直接求异

**定义：** 相同元素去掉，**不同元素留下**
- 主要针对**线条**

#### 3.4.2 旋转求异

**定义：** 图形**旋转** + **直接求异**

**步骤：**
1. 先将图形旋转
2. 再进行直接求异

## 四、样式类 VS 位置类 对比

| 类别      | 特征           | 考点         |
| :------ | :----------- | :--------- |
| **位置类** | 元素组成**完全相同** | 平移、旋转、翻转   |
| **样式类** | 元素组成**相似**   | 遍历、加法、求同求异 |

> 详见 [[图形推理-位置类]]

---

## 五、快速判断流程

```
观察图形
    │
    ▼
元素组成完全相同？ ──是──→ [[图形推理-位置类|位置类]]
    │否
    ▼
 样式类
    │
    ├── 元素种类和数量相同？ → 遍历（缺啥补啥）
    │
    ├── 有黑有白，数量相近？ → 黑白运算
    │
    ├── 线条相似，有叠加？ → 加减同异
    │   ├── 加法（叠加）
    │   ├── 求同（去异求同/整体求同）
    │   └── 求异（直接求异/旋转求异）
    │
    └── 其他样式变化
```

---

## 六、核心技巧汇总

| 考点 | 关键特征 | 解题方法 |
|:-----|:---------|:---------|
| **遍历** | 数量一致、种类一致 | 缺啥补啥 |
| **黑白运算** | 有黑有白，差值<3 | 位置对应，叠加计算 |
| **加减同异** | 线条相似，可叠加 | 直接运算/去异求同 |
| **求同** | 都有共同元素 | 取重叠/找共同 |
| **求异** | 去同存异 | 去掉相同，保留不同 |
