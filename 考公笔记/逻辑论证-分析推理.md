# 第五章 分析推理
## 5.1 真假话分析推理

|                常见提示词                 |   翻译    |
| :----------------------------------: | :-----: |
| 如果 A 那么 B，只要 A 就 B，一 A 就 B，所有 A 都是 B |  A → B  |
|      只有 A オ B、必须 A オ B、不 A 不 B       |  B → A  |
|      除非 A 否则 B，必须 A 否则 B，A 否则 B      | 非 A → B |
|    **B 是 A 的基础/关键/前提，A 离不开/需要B**     |  A → B  |

### 5.1.1 矛盾法
先把题干中每个命题的翻译写出来，观察是否有矛盾。
### 5.1.2 假设法
假设某选项成立，观察其成立是否影响其他选项的成立，导致违反了题干“只有一真”或“只有一假”
### 5.1.3 代入法
题干信息极其丰富，乱七八糟，没办法正面推出来，就用代入法
