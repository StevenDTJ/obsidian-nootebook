# 📚 错题本（按日期排序）

> 按日期从新到旧排列

---

## 错题列表

```dataview
TABLE WITHOUT ID
  date(date) as 日期,
  题型,
  模块,
  知识点,
  正确答案,
  错误原因
FROM "错题"
SORT date DESC
```
