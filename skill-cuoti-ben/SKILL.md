# 错题本 Skill

## 核心功能

1. **增加错题** - 将新错题存为独立文件（`错题/序号.md`）
2. **错题重考** - 随机抽取错题进行练习
3. **自动统计** - Dataview自动汇总到错题本和仪表盘

## 文件结构

```
vault/
├── 错题本.md          # 索引文件（Dataview自动汇总）
├── 错题仪表盘.md      # 统计仪表盘（自动更新）
└── 错题/              # 错题文件夹
    ├── 1.md
    ├── 2.md
    ├── 3.md
    └── ...
```

## 错题文件模板

每道错题存为 `错题/X.md`，格式如下：

```markdown
---
date: 2026-03-04
题型: 选择题
模块: 图形推理
知识点: 位置类-平移
正确答案: A
错误原因: 未发现规律
---

# 错题 X：[标题]

![错题](../image/图片名.png)

**正确答案：** A

**详细解答：**
[解题步骤/思路]

**错误原因：** [当时为什么做错]
```

## 添加错题流程

1. **收集信息**：向用户获取以下内容：
   - 错题图片（用户粘贴或提供文件）
   - 题型（选择题/填空题/解答题等）
   - 模块（如：图形推理、言语理解等）
   - 知识点（如：位置类-平移、中心理解等）
   - 正确答案
   - 详细解答
   - 错误原因（必填，不可省略）
   - 如未提供日期，使用当天日期

2. **缺失信息处理**：
   - 如果用户**未提供错误原因**，必须向用户确认：
     > "请问错误原因是？如暂无总结，可回复'略'"
   - 若用户回复"略"或类似表述，则错误原因填写"略"
   - 若用户不回复或拒绝填写，**暂不添加该错题**，等待用户提供完整信息

3. **处理图片**：
   - 下载用户图片到 `image/` 文件夹
   - 命名格式：`cuoti_tupian_N.png`（N为序号）

4. **确定序号**：
   - 列出 `错题/` 文件夹中的所有.md文件
   - 新序号 = 最大序号 + 1

5. **创建错题文件**：
   - 在 `错题/` 文件夹创建新文件
   - 使用上面的模板格式
   - frontmatter中的字段：date, 题型, 模块, 知识点, 正确答案, 错误原因

6. **更新索引**：Dataview会自动更新 `错题本.md` 和 `错题仪表盘.md`

## 错题重考流程

1. **读取错题列表**：使用Dataview查询 `错题/` 文件夹
2. **随机抽取**：
   - 询问用户要抽取几道题（默认3道）
   - 随机选择对应数量的错题
3. **展示题目**：只显示题目和图片，不显示答案
4. **用户作答**：用户给出答案后
5. **验证答案**：显示正确答案，让用户核对

## 使用示例

### 添加错题
```
用户：把这道数学题加到错题本
用户：![粘贴图片]
用户：答案是B，模块是图形推理，知识点是位置类-平移，错因是忘记考虑定义域
```

### 错题重考
```
用户：来一次错题重考
用户：抽3道题
```

## 注意事项

- 图片存放在 `image/` 文件夹
- 错题文件存放在 `错题/` 文件夹
- frontmatter字段：`date`, `题型`, `模块`, `知识点`, `正确答案`, `错误原因`
- `错题本.md` 是索引文件，显示所有错题列表
- `错题仪表盘.md` 是统计仪表盘，自动更新统计数据
