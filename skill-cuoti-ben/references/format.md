# 错题本格式参考

## 文件位置

- 错题文件夹：`错题/`
- 图片文件夹：`image/`
- 索引文件：`错题本.md`
- 统计仪表盘：`错题仪表盘.md`

## 错题文件模板

```markdown
---
date: 2026-03-04
题型: 选择题
模块: 图形推理
知识点: 位置类-平移
正确答案: A
错误原因: 未发现规律
---

# 错题 X：[标题]

![错题](../image/图片名.png)

**正确答案：** A

**详细解答：**
[解题步骤/思路]

**错误原因：** [当时为什么做错]
```

## Frontmatter字段说明

| 字段 | 必填 | 说明 | 示例 |
|:-----|:----:|:-----|:-----|
| date | 是 | 日期 | 2026-03-04 |
| 题型 | 是 | 题型 | 选择题、填空题、解答题 |
| 模块 | 是 | 学科模块 | 图形推理、言语理解、数量关系 |
| 知识点 | 是 | 具体知识点 | 位置类-平移、中心理解 |
| 正确答案 | 是 | 正确答案 | A、B、C、D 或 具体答案 |
| 错误原因 | 是 | 错误原因 | 未发现规律、计算错误 |

## 模块分类（参考）

- 图形推理
- 定义判断
- 类比推理
- 逻辑判断
- 言语理解
- 数量关系
- 资料分析
- 常识判断

## 知识点分类示例

### 图形推理
- 位置类：平移、旋转、翻转
- 样式类：遍历、加减同异、黑白运算
- 属性类：对称性、曲直性、开闭性
- 数量类：点、线、角、面、素

### 言语理解
- 中心理解
- 细节判断
- 词语理解
- 句子排序

## 错误原因分类（参考）

- 概念理解错误
- 公式记忆错误
- 计算失误
- 审题不清
- 漏掉条件
- 方法不当
- 粗心大意
- 完全没有思路
- 未发现规律
