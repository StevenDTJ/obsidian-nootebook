# 三次工业革命对比

## 第一次工业革命

| 项目 | 内容 |
|:----:|:-----|
| **主要标志** | 蒸汽机的改良 |
| **理论基础** | 牛顿力学（惯性定律、加速度、作用力与反作用力） |
| **领先国家** | 英国 |
| **时代** | 蒸汽时代 |
| **交通工具** | 汽船、火车 |
| **主要发明** | 瓦特改良蒸汽机、富尔顿发明汽船、斯蒂芬孙发明火车 |

---

## 第二次工业革命

| 项目 | 内容 |
|:----:|:-----|
| **主要标志** | 电力的广泛应用 |
| **理论基础** | 法拉第的电磁感应学 |
| **领先国家** | 美国、德国 |
| **时代** | 电气时代 |
| **交通工具** | 电车、汽车、飞机 |
| **主要发明** | 爱迪生电灯、西门子发电机、贝尔电话、卡尔·本茨汽车、莱特兄弟飞机 |

---

## 第三次工业革命

| 项目 | 内容 |
|:----:|:-----|
| **主要标志** | 原子能、电子计算机、航天工程、生物技术 |
| **理论基础** | 爱因斯坦相对论 |
| **领先国家** | 美国 |
| **时代** | 网络信息时代 |
| **新兴产业** | 信息技术、核工业、航空航天、电子工业 |
| **主要发明** | 原子弹、氢弹 |
