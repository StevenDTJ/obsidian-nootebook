# 空气组成

空气是混合物，主要成分：

| 成分 | 体积占比 | 性质与用途 |
|:----:|:--------:|:-----------|
| **氮气** | 78% | 惰性保护气、制造硝酸和化肥、液氮冷冻 |
| **氧气** | 21% | 供给呼吸、支持燃烧、炼钢 |
| 稀有气体 | 0.94% | 保护气、电光源 |
| **二氧化碳** | 0.03% | 不支持燃烧（灭火）、不能供给呼吸 |
| 其他 | 0.03% | 杂质 |

## 常见气体

| 气体 | 特性 |
|:----:|:-----|
| 甲烷（天然气/沼气主要成分） | 无毒，但浓度10%可致窒息 |
| 一氧化碳 | **煤气中毒元凶**，煤灶放水不能防中毒 |
| 氧气 | 燃烧条件：达到着火点 + 有助燃剂 |
| 干冰 | 固态CO₂，用于制冷和人工降雨 |

> [!warning] 灭火：热水比冷水效果好（热水形成蒸汽，隔绝氧气）
