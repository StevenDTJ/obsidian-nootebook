# 八大行星

太阳系八大行星，按距离太阳由近到远排列：

> **口诀**：**水金地火木土天海**

| 行星 | 顺序 | 特点 |
|:----:|:----:|:-----|
| 水星 | 第1 | 离太阳最近，温差最大 |
| 金星 | 第2 | 逆向自转，「启明星」或「长庚星」 |
| 地球 | 第3 | 唯一有生命存在的星球 |
| 火星 | 第4 | 「红色星球」，有季节变化 |
| 木星 | 第5 | 体积最大，有大红斑 |
| 土星 | 第6 | 有美丽的光环 |
| 天王星 | 第7 | 躺着自转 |
| 海王星 | 第8 | 距离太阳最远 |
