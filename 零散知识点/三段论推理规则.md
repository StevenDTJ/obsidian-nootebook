[[考公笔记/推理方式与论证结构|三段论]]是由两个前提推出一个结论的演绎推理，考公常考以下三条基本推导规则。

**规则一：所有 + 所有 = 所有**
> 所有 A 是 B，所有 B 是 C → 所有 A 是 C
> 例：所有人都会死，苏格拉底是人 → 苏格拉底会死

**规则二：所有 + 有些 = 有些**
> 所有 A 是 B，有些 C 是 A → 有些 C 是 B
> 例：所有鸟都有翅膀，有些动物是鸟 → 有些动物有翅膀
> ⚠️ 注意方向：必须是"有些"出现在包含于"所有"的那一侧才能推出结论

**规则三：所有 + 没有 = 没有**
> 所有 A 是 B，没有 B 是 C → 没有 A 是 C
> 例：所有玫瑰是花，没有花是石头 → 没有玫瑰是石头

**做题口诀**：
- 两个"有些"不能推出任何必然结论
- 两个否定不能推出任何结论
- 有一个否定前提，结论必须是否定的
