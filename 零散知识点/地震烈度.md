# 地震烈度

**震级**：地震大小，**一次地震只有一个震级**（用字母M表示）

**烈度**：破坏程度，**一次地震可以有多个烈度**

> **震级每相差1.0级，能量相差约30倍**

## 中国地震烈度表

| 烈度 | 等级 | 典型表现 |
|:----:|:----:|:--------|
| I-II | 微/小震 | 仅仪器可记录 |
| III-IV | 小震 | 少数人有感，吊物摇晃 |
| V | 中小震 | 睡觉惊醒，物品掉落 |
| VI | 中震 | 树木摇动，房屋轻微损害 |
| VII-VIII | 大震 | 房屋掉土、墙裂 |
| IX-X | 大地震 | 地裂、喷水喷沙、建筑物多数倒塌 |
| XI-XII | 特大地震 | 全面破坏，地面起伏如波浪 |

## 地震自救要点

1. 切断电源、气源，防止火灾
2. 逃到室外空旷处
3. **避开**：高大建筑物、窄小胡同、陡山坡、河岸边
4. 来不及时：承重墙墙根、墙角、有水管和暖气管道处
