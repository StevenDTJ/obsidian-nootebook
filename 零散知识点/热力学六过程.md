# 热力学六过程

物质三态之间的六种变化：

| 过程 | 变化方向 | 吸/放热 |
|:----:|:--------:|:-------:|
| **汽化** | 液态 → 气态 | 吸热 |
| **液化** | 气态 → 液态 | 放热 |
| **熔化** | 固态 → 液态 | 吸热 |
| **凝固** | 液态 → 固态 | 放热 |
| **升华** | 固态 → 气态 | 吸热 |
| **凝华** | 气态 → 固态 | 放热 |

> **记忆口诀**：「溶化、汽化升华要吸热，其余放热」

> [!tip] 区分：蒸发和沸腾是汽化的两种方式；干冰（固态CO2）升华是常见的生活现象
