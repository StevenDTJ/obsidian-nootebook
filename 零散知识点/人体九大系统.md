# 人体九大系统

人体由九大系统组成，构成完整的人体结构。

| 系统 | 主要组成 | 常见疾病 |
|:----:|:--------:|:--------:|
| 运动系统 | 骨、关节、骨骼肌 | 肩周炎、骨质增生、佝偻病、骨质疏松 |
| 消化系统 | 消化道（口→肛）+ 消化腺（胰腺、肝脏等） | 肝胆疾病、消化道溃疡、肠胃炎 |
| 呼吸系统 | 呼吸道、肺 | 肺炎、支气管炎、哮喘、感冒 |
| 循环系统 | 心脏、血管 | 冠心病、高血压、心绞痛 |
| 神经系统 | 中枢（脑、脊髓）+ 周围神经 | 癫痫、老年痴呆 |
| 内分泌系统 | 甲状腺、垂体、肾上腺、胰岛 | 甲亢、糖尿病 |
| 免疫系统 | 免疫器官（骨髓、脾脏）+ 免疫细胞 | 艾滋病、风湿性关节炎 |
| 泌尿系统 | 肾、膀胱、尿道 | 肾炎、尿路结石 |
| 生殖系统 | 生殖腺、生殖管道 | 妇科/男科疾病 |

> **记忆**：口诀「神循呼消动内免泌生」
