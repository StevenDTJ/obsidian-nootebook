# 生物圈组成

## 生态系统的组成

| 成分 | 代表 | 作用 |
|:----:|:----:|:-----|
| **生产者** | 绿色植物 | 通过光合作用将无机物合成为有机物 |
| **消费者** | 动物（包括人） | 消耗有机物 |
| **分解者** | 微生物（细菌、真菌） | 将有机物分解为无机物 |

> [!important] **结构层次**：细胞 → 组织 → 器官 → 系统 → 个体 → 种群 → 群落 → 生态系统 → 生物圈

## 生物进化顺序

### 动物进化
无脊椎动物 → 鱼类 → 两栖类 → 爬行类 → **鸟类/哺乳类**

### 植物进化
藻类 → 苔藓类 → 蕨类 → 裸子植物 → 被子植物

### 人类进化
猿人 → 直立人 → 智人

> **记忆**：总体趋势是**由简单到复杂、由低等到高等、由水生到陆生**
