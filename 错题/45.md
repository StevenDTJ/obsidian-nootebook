# 错题 45：判断推理-食育行动

**来源**：行测定义判断题

![错题](../image/cuoti_45.png)

<details>
<summary>点击查看答案</summary>

<b>你的答案</b>：D<br>
<b>正确答案</b>：C<br><br>
<b>详细解答</b>：<br>
<strong>食育行动</strong>定义：通过<strong>科学的营养知识普及活动</strong>，以适合一国国情的方式，让<strong>居民</strong>渐渐养成良好的饮食生活习惯。

<strong>关键要素：</strong>
<ol>
<li><strong>科学的营养知识普及活动</strong>（科普性质，非专业教育）</li>
<li>对象是<strong>居民</strong>（面向大众，非特定群体如学生）</li>
<li>目的是养成<strong>良好的饮食生活习惯</strong></li>
</ol>

<strong>选项分析：</strong>
<ul>
<li><strong>A项</strong>：烹饪学校要求学生安全使用工具处理食材——针对<strong>特定群体（学生）</strong>的专业技能要求，非面向居民的科普活动，不符合定义。</li>
<li><strong>B项</strong>：社区开展体育锻炼活动——主题是<strong>体育锻炼</strong>，非营养知识普及，不符合定义。</li>
<li><strong>C项</strong>：有关部门颁布《国民营养计划(2017-2030年)》——针对<strong>本国国民</strong>的饮食生活，通过政策推动营养知识普及，让居民养成良好饮食习惯，<strong>符合定义</strong>。</li>
<li><strong>D项</strong>：食品学院培养学生分辨食材——针对<strong>特定群体（学生）</strong>的专业教育，非面向居民的科普活动，不符合定义。</li>
</ul>

<strong>错误原因</strong>：认为C不算科普活动，忽略了"主题相关"这一限定词。实际上，《国民营养计划》是针对国民的宏观营养政策，属于面向居民的营养知识普及活动；而D项是针对学生的专业教育，对象不符合"居民"的要求。<br><br>

<strong>核心要点</strong>：
<ul>
<li>食育行动的<strong>对象必须是"居民"</strong>（大众），而非特定群体如学生</li>
<li>学校的专业教育（烹饪学校、食品学院）不属于面向居民的科普活动</li>
<li>政策层面的营养计划属于面向全民的科普活动</li>
<li>必须紧扣"<strong>饮食生活习惯</strong>"这一主题，体育锻炼不符合</li>
</ul>
</details>
